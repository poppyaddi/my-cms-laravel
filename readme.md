## About
基础后台模板，只包含系统管理，其子模块如下：
- 用户管理
- 用户详情
- 角色管理
- 菜单管理
- 权限管理

## 配置安装
1. 下载项目
2. `composer install`安装依赖
3. 执行`php artisan key:generate`
4. 执行`php artisan jwt:secret`
